# Backend Documentation

**IAC-011 Sovereign Backend** - Shared FastAPI foundation for multiple projects

## Structure

- **`pm/`** - Project management and planning documents
- **`dev/`** - Development guides and API documentation

## About This Module

The Backend module provides shared services for:
- FastAPI REST API endpoints
- PostgreSQL + pgvector database layer
- Redis + Celery job queues
- ChromaDB RAG integration
- Platform-specific scrapers (Twitter, YouTube, Reddit)
