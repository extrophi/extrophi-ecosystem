                        HOW TO USE

  1. Launch the app: npm run tauri:dev
  2. Press Cmd+? (or Ctrl+?) to see the shortcuts help
  3. Try Cmd+R to start recording
  4. Navigate views with Cmd+1/2/3/4
  5. Click sessions list and use ↑↓ arrows to navigate
