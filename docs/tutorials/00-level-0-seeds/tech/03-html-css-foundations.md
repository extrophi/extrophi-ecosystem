# Tutorial Outline: HTML/CSS Foundations

**Category**: Tech
**Level**: 0 (Seed)
**Estimated Time**: 8-10 hours

## Learning Objectives

- Build semantic, accessible HTML documents from scratch
- Style web pages using modern CSS techniques
- Understand the box model and layout fundamentals
- Create responsive designs that work across devices
- Debug HTML/CSS issues using browser developer tools

## Section 1: HTML Document Structure

**Concepts**:
- What is HTML: markup vs programming
- Document Object Model (DOM)
- Semantic markup and accessibility
- HTML5 document structure

**Skills**:
- Create valid HTML5 documents with DOCTYPE
- Use proper document structure (html, head, body)
- Add metadata (title, charset, viewport)

## Section 2: Essential HTML Elements

**Concepts**:
- Block vs inline elements
- Headings hierarchy (h1-h6)
- Paragraphs, lists, and text formatting
- Links and navigation

**Skills**:
- Structure content with semantic elements
- Create ordered and unordered lists
- Add links with proper href attributes

## Section 3: Semantic HTML5 Elements

**Concepts**:
- Semantic vs div/span soup
- Structural elements (header, nav, main, footer)
- Content grouping (article, section, aside)
- Accessibility benefits of semantic HTML

**Skills**:
- Use semantic elements appropriately
- Structure page layouts semantically
- Choose correct element for content type

## Section 4: Forms and Input

**Concepts**:
- Form submission and HTTP methods
- Input types and attributes
- Form validation (HTML5)
- Labels and accessibility

**Skills**:
- Create functional forms
- Use various input types (text, email, checkbox, radio)
- Implement basic client-side validation

## Section 5: CSS Fundamentals

**Concepts**:
- Selectors: element, class, ID, attribute
- Specificity and cascade
- Inheritance
- CSS properties and values

**Skills**:
- Write CSS selectors of varying specificity
- Apply styles using classes and IDs
- Use browser DevTools to inspect styles

## Section 6: Box Model and Layout

**Concepts**:
- Content, padding, border, margin
- Box-sizing property
- Display property (block, inline, inline-block)
- Width, height, and sizing

**Skills**:
- Control spacing with margin and padding
- Understand box model calculations
- Debug layout issues with DevTools

## Section 7: Flexbox Layout

**Concepts**:
- Flex container and flex items
- Main axis vs cross axis
- Flex properties (justify-content, align-items)
- Flex-direction and flex-wrap

**Skills**:
- Create flexible layouts with flexbox
- Center elements horizontally and vertically
- Build navigation bars and card layouts

## Section 8: Responsive Design Basics

**Concepts**:
- Mobile-first design philosophy
- Media queries and breakpoints
- Viewport units (vw, vh)
- Responsive images

**Skills**:
- Write media queries for different screen sizes
- Create mobile-friendly layouts
- Test responsive designs in DevTools

## Capstone Project

**Personal Portfolio Website**

Build a responsive, accessible portfolio site with:

1. **Homepage**:
   - Hero section with introduction
   - Navigation bar
   - Featured projects grid (3-4 items)
   - Contact section with form

2. **About Page**:
   - Biography with semantic markup
   - Skills list
   - Timeline or experience section

3. **Project Detail Page**:
   - Project description
   - Images/screenshots
   - Technologies used
   - Link to live demo/code

**Technical Requirements**:
- Valid HTML5 with semantic elements
- CSS-only design (no frameworks)
- Flexbox for layouts
- Responsive design (mobile, tablet, desktop)
- Accessible forms with proper labels
- No browser console errors

**Deliverables**:
- HTML/CSS source code
- Working website viewable in browser
- README with design decisions
- Screenshots at different breakpoints

## Prerequisites

- Basic computer literacy
- Text editor (VS Code recommended)
- Modern web browser (Chrome/Firefox)
- Understanding of file systems and folders
- Willingness to experiment and iterate

## Next Steps

After completing this tutorial, you should be ready for:
- **CSS Grid Layout**: Advanced 2D layouts
- **JavaScript Essentials**: Adding interactivity
- **Web Accessibility (A11y)**: WCAG compliance, ARIA
- **CSS Preprocessors**: Sass/SCSS
- **Web Design Fundamentals**: Typography, color theory, spacing
- **Modern CSS**: Custom properties, animations, transitions

## Resources

- MDN Web Docs (Mozilla Developer Network)

- CSS-Tricks (flexbox guide, layout patterns)
- A11y Project (accessibility checklist)
- Can I Use (browser compatibility reference)

## Assessment Criteria

Students should be able to:
- [ ] Write semantic, valid HTML5 documents
- [ ] Apply CSS styles using appropriate selectors
- [ ] Explain and demonstrate the box model
- [ ] Create responsive layouts with flexbox
- [ ] Use browser DevTools for debugging
- [ ] Build accessible forms with proper markup
- [ ] Complete capstone project meeting all requirements
