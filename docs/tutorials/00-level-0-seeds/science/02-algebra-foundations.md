# Tutorial Outline: Algebra Foundations

**Category**: Science
**Level**: 0 (Seed)
**Estimated Time**: 10-12 hours

## Learning Objectives

- Master algebraic manipulation and equation solving
- Understand functions and their representations
- Apply algebra to model real-world situations
- Build foundation for advanced mathematics
- Develop algebraic reasoning and pattern recognition

## Section 1: Variables and Expressions

**Concepts**:
- Variables as unknowns and generalizations
- Algebraic expressions vs equations
- Evaluating expressions
- Order of operations with variables

**Skills**:
- Write algebraic expressions
- Evaluate expressions for given values
- Simplify expressions

## Section 2: Equation Solving

**Concepts**:
- Equality and maintaining balance
- Inverse operations
- Linear equations in one variable
- Checking solutions

**Skills**:
- Solve linear equations
- Isolate variables systematically
- Verify solutions

## Section 3: Systems of Equations

**Concepts**:
- Multiple equations, multiple unknowns
- Substitution method
- Elimination method
- Graphical interpretation

**Skills**:
- Solve systems of 2 equations
- Choose appropriate method
- Interpret solutions graphically

## Section 4: Functions

**Concepts**:
- Function definition and notation
- Domain and range
- Linear, quadratic, exponential functions
- Function transformations

**Skills**:
- Identify functions from tables, graphs, equations
- Evaluate functions
- Graph basic functions

## Section 5: Polynomials

**Concepts**:
- Polynomial definition and degree
- Adding, subtracting, multiplying polynomials
- Factoring techniques
- Quadratic formula

**Skills**:
- Perform polynomial operations
- Factor polynomials
- Solve quadratic equations

## Section 6: Inequalities

**Concepts**:
- Inequality symbols and meaning
- Solving inequalities
- Compound inequalities
- Graphing solution sets

**Skills**:
- Solve linear inequalities
- Graph inequalities on number line
- Interpret inequality solutions

## Section 7: Exponentials and Logarithms

**Concepts**:
- Exponential growth and decay
- Logarithms as inverse of exponentials
- Properties of logarithms
- Exponential equations

**Skills**:
- Work with exponential expressions
- Solve exponential equations
- Use logarithms for solving

## Section 8: Applications and Modeling

**Concepts**:
- Word problems and translation
- Rate, distance, time problems
- Mixture and work problems
- Optimization with algebra

**Skills**:
- Translate word problems to algebra
- Model real situations
- Solve applied problems

## Capstone Project

**Algebra Mastery Portfolio**

Comprehensive demonstration of algebraic skills:

1. **Equation Solving Mastery** (50+ problems):
   - Linear equations (one variable)
   - Systems of equations (2-3 variables)
   - Quadratic equations
   - Exponential equations
   - Radical equations

   Show all work and verify solutions

2. **Function Analysis** (10+ functions):
   - For each function:
     - Graph
     - Domain and range
     - Intercepts
     - Key features
     - Transformations
     - Applications

3. **Polynomial Project**:
   - Factor 20+ polynomials
   - Multiply complex expressions
   - Solve polynomial equations
   - Real-world polynomial models
   - Graphing polynomial functions

4. **Word Problem Collection** (20+ problems):
   - Age problems
   - Distance/rate/time
   - Work problems
   - Mixture problems
   - Optimization problems
   - Finance and growth

   For each: variables defined, equation setup, solution, interpretation

5. **Real-World Modeling Project**:
   - Choose phenomenon to model
   - Collect or find data
   - Build algebraic model (linear, quadratic, or exponential)
   - Analyze and make predictions
   - Validate model accuracy
   - Document full process

6. **Algebra Reference Guide**:
   - Key formulas and techniques
   - Common mistakes to avoid
   - Problem-solving strategies
   - Practice problems with solutions
   - Visual aids and examples

**Deliverables**:
- 50+ solved equations showing work
- 10 function analyses
- Polynomial operations portfolio
- 20+ word problems solved
- Real-world modeling project
- Personal algebra reference guide
- Reflection on learning journey

## Prerequisites

- Mathematical Thinking basics
- Arithmetic proficiency
- Basic fraction/decimal operations
- Persistence with abstract thinking

## Next Steps

After completing this tutorial, you should be ready for:
- **Precalculus**: Trigonometry, advanced functions
- **Calculus**: Limits, derivatives, integrals
- **Linear Algebra**: Matrices, vector spaces
- **Discrete Math**: Combinatorics, sequences
- **Statistics**: Data analysis with algebra
- **Physics**: Algebra-based physics
