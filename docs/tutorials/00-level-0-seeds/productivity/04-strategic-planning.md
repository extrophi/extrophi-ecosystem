**Level**: 0 (Seed)
**Estimated Time**: 6-8 hours

## Capstone Project

**Annual Life Plan and Execution**

Create and execute comprehensive strategic plan:

1. **Life Vision and Values** (Foundation):
   - Personal mission statement
   - Core values identification (5-7 values)
   - Life areas definition (8-10 areas)
   - 10-year vision by area
   - Eulogy/legacy exercise

2. **Annual Strategic Plan** (Year-level):
   - **Annual Goals** (3-5 major goals):
     - Career/professional
     - Health/fitness
     - Relationships/family
     - Learning/growth
     - Finance/wealth

   - For each goal:
     - SMART specification
     - Why this goal matters (motivation)
     - Success criteria and metrics
     - Potential obstacles
     - Required resources

3. **Quarterly Planning** (Q1 execution):
   - **Q1 OKRs** for each annual goal
   - Objectives (qualitative)
   - Key Results (quantitative, 3-5 per objective)
   - Monthly milestones
   - Weekly breakdown

4. **Project Plans** (3-5 major projects):
   - Project charters (goals, scope, timeline)
   - Work breakdown structure
   - Gantt chart or timeline
   - Resource and budget planning
   - Risk assessment

5. **Execution and Tracking** (12 weeks):

   **Daily**:
   - Morning: Day planning with goals in mind
   - Evening: Progress logging and reflection

   **Weekly**:
   - Sunday review and planning
   - Progress tracking dashboard update
   - Wins and lessons documentation

   **Monthly**:
   - Monthly review against OKRs
   - Course corrections
   - Next month planning

   **Quarterly** (end of Q1):
   - Full OKR review
   - Q2 planning
   - Annual plan adjustment

6. **Dashboard and Visualization**:
   - Goal tracking dashboard
   - Progress charts and graphs
   - Habit trackers
   - Milestones and celebrations
   - Photos/evidence of progress

**Deliverables**:
- Life vision and values document
- Annual strategic plan (5+ goals)
- Q1 OKR documentation
- 3-5 project plans
- 12-week execution logs (daily, weekly, monthly)
- Progress dashboard
- Quarterly review report
- Reflection on goal achievement
