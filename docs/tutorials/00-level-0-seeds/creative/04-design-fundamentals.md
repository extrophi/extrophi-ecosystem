# Tutorial Outline: Design Fundamentals

**Category**: Creative
**Level**: 0 (Seed)
**Estimated Time**: 10-12 hours

## Learning Objectives

- Understand core design principles and apply them effectively
- Develop visual design skills for digital and print
- Create designs with intention and purpose
- Use typography, color, and layout confidently
- Build foundational design literacy

## Section 1: Design Thinking and Process

**Concepts**:
- What is design (problem-solving, not decoration)
- Design thinking process
- Form follows function
- User-centered design

**Skills**:
- Approach design as problem-solving
- Define design goals and constraints
- Critique design intentionally

## Section 2: Core Design Principles

**Concepts**:
- Contrast, repetition, alignment, proximity (CRAP)
- Visual hierarchy
- Balance and symmetry
- Unity and harmony

**Skills**:
- Apply design principles consciously
- Create visual hierarchy
- Achieve balanced compositions

## Section 3: Typography Basics

**Concepts**:
- Typeface vs font
- Serif, sans-serif, display, monospace
- Type anatomy and terminology
- Readability vs legibility

**Skills**:
- Choose appropriate typefaces
- Pair fonts effectively
- Set type for readability

## Section 4: Color Theory

**Concepts**:
- Color wheel and relationships
- Hue, saturation, value
- Color psychology and meaning
- Accessibility and contrast

**Skills**:
- Create color palettes
- Use color intentionally
- Ensure sufficient contrast

## Section 5: Layout and Composition

**Concepts**:
- Grid systems
- White space and breathing room
- Rule of thirds and golden ratio
- Visual flow and reading order

**Skills**:
- Use grids for structure
- Balance content with white space
- Guide viewer through design

## Section 6: Visual Communication

**Concepts**:
- Icons and symbols
- Photography and imagery
- Illustration vs photography
- Visual metaphors

**Skills**:
- Select appropriate imagery
- Use icons effectively
- Create visual metaphors

## Section 7: Design Tools and Workflow

**Concepts**:
- Design software landscape (Figma, Adobe, Canva)
- Layers and organization
- Export formats and optimization
- Design systems and consistency

**Skills**:
- Use design software basics
- Organize design files
- Export for various uses

## Section 8: Design Critique and Iteration

**Concepts**:
- Giving and receiving feedback
- Iteration and refinement
- Testing and validation
- When to stop designing

**Skills**:
- Critique designs constructively
- Iterate based on feedback
- Recognize "good enough"

## Capstone Project

**Design Portfolio: 10 Projects**

Create diverse design projects demonstrating principles:

1. **Brand Identity**:
   - Logo design (wordmark and icon)
   - Color palette (5 colors with purpose)
   - Typography system (3 fonts)
   - Brand guidelines (one-page)

2. **Poster/Flyer Design**:
   - Event poster (concert, conference, or workshop)
   - Clear hierarchy and focal point
   - Typography-focused design
   - Print-ready (300 DPI, CMYK)

3. **Social Media Graphics** (5-set):
   - Consistent template design
   - Quote graphics
   - Announcement cards
   - Profile graphics
   - Optimized for platform

4. **Presentation Slides** (10 slides):
   - Cohesive visual system
   - Minimal text, strong visuals
   - Data visualization (chart/graph)
   - Title, content, and closing slides

5. **One-Page Website Design**:
   - Hero section
   - Features/benefits
   - Call-to-action
   - Mobile-responsive mockup

6. **Infographic**:
   - Data-driven visual story
   - Clear visual hierarchy
   - Icons and illustrations
   - Scannable and shareable

7. **Business Card**:
   - Front and back design
   - Professional and memorable
   - Print specifications
   - Die-cut or special finish concept

8. **Icon Set** (20 icons):
   - Consistent style
   - Multiple sizes
   - Accessibility considerations
   - Light and dark versions

9. **Typography Poster**:
   - Type-only design
   - Demonstrate hierarchy
   - Single or paired fonts
   - Experimental and expressive

10. **Redesign Challenge**:
    - Choose poorly designed item
    - Critique original
    - Redesign with improvements
    - Document design decisions

**Additional Requirements**:
- **Design System Documentation**: Color, type, spacing standards
- **Process Documentation**: Sketches, iterations, rationale
- **Presentation**: Portfolio website or PDF
- **Reflection**: What you learned, strengths, growth areas

**Deliverables**:
- 10 completed design projects
- Process documentation for each
- Design system style guide
- Portfolio presentation
- Reflection essay (1500+ words)

## Prerequisites

- Visual Thinking or Sketching helpful
- No design experience required
- Design software access (Figma free tier sufficient)
- Willingness to iterate and receive feedback
- Attention to detail

## Next Steps

After completing this tutorial, you should be ready for:
- **UI/UX Design**: Interface design, user experience
- **Graphic Design Advanced**: Advanced composition, trends
- **Brand Design**: Comprehensive brand identity systems
- **Motion Graphics**: Animation and video design
- **Print Design**: Advanced print production
- **Web Design**: HTML/CSS implementation of designs

## Resources

- **Books**:
  - "The Non-Designer's Design Book" by Robin Williams
  - "Thinking with Type" by Ellen Lupton
  - "The Elements of Graphic Design" by Alex W. White
  - "Refactoring UI" by Adam Wathan & Steve Schoger

- **Tools**:
  - Figma (free for individuals)
  - Canva (easy for beginners)
  - Adobe Creative Suite (industry standard)
  - Affinity Designer/Photo (one-time purchase)

- **Resources**:
  - Dribbble, Behance (inspiration)
  - Google Fonts (free typography)
  - Unsplash, Pexels (free photos)
  - Coolors, Adobe Color (color palettes)

- **Learning**:
  - Refactoring UI (practical tips)
  - Laws of UX (design principles)
  - Design Better podcast

## Assessment Criteria

Students should be able to:
- [ ] Apply core design principles (contrast, alignment, etc.)
- [ ] Create effective visual hierarchy
- [ ] Use typography and color intentionally
- [ ] Design for specific audiences and purposes
- [ ] Use design software competently
- [ ] Give and receive design critique
- [ ] Complete 10-project portfolio
- [ ] Articulate design decisions clearly
